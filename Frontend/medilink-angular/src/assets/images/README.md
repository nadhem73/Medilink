# 📸 Images - Smart Health Tunisia

## Structure des Dossiers

```
images/
├── banner/          → Fond hero banner (hôpital tunisien, Méditerranée)
├── users/           → Avatars patients (rating badge)
├── partners/        → Logos partenaires tunisiens (CNAM, Ministère, etc.)
├── services/        → Photos services (téléconsultation, pharmacie, labo, ambulance)
├── doctors/         → Photos médecins tunisiens
└── testimonials/    → Avatars témoignages patients
```

## 🇹🇳 Sources d'Images Tunisiennes Recommandées

### Images Médicales Tunisiennes:
1. **Unsplash** - Rechercher: "Tunisia hospital", "Tunis medical"
2. **Pexels** - Rechercher: "Tunisia healthcare", "Mediterranean clinic"
3. **Pixabay** - Images libres de droits
4. **Photos personnelles** - Avec autorisations nécessaires

### Partenaires Tunisiens à Contacter:
- Ministère de la Santé Tunisie (logo officiel)
- CNAM (Caisse Nationale d'Assurance Maladie)
- Ordre National des Médecins de Tunisie
- Pharmacie Centrale de Tunisie
- Hôpitaux publics tunisiens

## 📋 Spécifications Techniques

| Type | Dimensions | Format | Poids Max |
|------|-----------|--------|-----------|
